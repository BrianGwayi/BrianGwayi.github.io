About

This project demonstrates the implementation of a modern data stack and the development of a robust data pipeline for a retail company. It leverages a combination of Python for writing elt/etl scripts, Apache Airflow for orchestration, BigQuery, Snowflake, AWS Redshift for storage, dbt for transformation and Amazon Managed Workflow for Airiflow for production to efficiently acquire and move data from PostgreSQL database to an OLAP system.

Apache Airflow Local Setup

                    
# Apache Airflow - Local Set-Up
$ python3 -m venv airflow-env  # create virtual environment
$ source airflow-env/bin/activate  # activate virtual environment
$ export AIRFLOW_HOME=~/airflow 
$ pip install apache-airflow
$ airflow db init
$ airflow webserver -p 8080 #launch webserver
$ airflow sheduler #launch scheduler
                        
                    
                        
# installing dependencies
pip install google-cloud-bigquery
pip install --upgrade snowflake-connector-python
